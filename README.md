## 关于 阑珊

阑珊影视(node.js版)是个人最近学习 vue.js + node.js 而开发的一个练手项目，没有包装。

-|-|-
require模块|说明
express|替代http模块，提供web访问服务；提供静态文件加载功能;
body-parser|提供解析服务
puppeteer|提供web请求服务

## 食用方法

1.切换到代码目录，执行命令 node server.js   
2.访问地址：localhost 
3.输入你想看的影片，点击搜索, have fun ! [!点击查看图片](./screenshot/show3.png)
